# Problem 03: Sistem Pengiriman Komoditas Antar Desa (Python)

## 1. Deskripsi Permasalahan

Permasalahan ekspedisi pengantaran barang hasil bumi dan komoditi desa yang dikelola oleh Perangkat masyarakat desa. Setiap desa memiliki beragam hasil produksi komoditas unggulan desa. Selain digunakan untuk keperluan pribadi beberapa komoditas ini dijual ke desa lainnya. 

Pemilihan rute terpendek ketika proses pengantaran komoditi diharapkan dapat menjadikan operasional distribusi barang menjadi lebih murah. 

Hal tersebut dapat menguntungkan kedua belah pihak, baik desa produsen komoditas dan desa mitra selaku konsumen tujuan pengiriman.

Bagaimana kira-kira solusi pemilihan dan penentuan jalan-jalan yang akan dilewati ketika proses pengiriman barang dari desa produsen menuju sebuah desa konsumen mitranya ini?
